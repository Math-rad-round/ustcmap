课程json格式
{
    "id","name","content","symbols",
    "options":[
        {"name","content","cost"
            "effects":[
                {一系列下列的要求

                }
            ]
        },
        {.....},
    ]
}

effects列表

{getVariable,target,output},target取theory/practice/social/semester/achievement等值
{addNextScore,input,(multi=1)},增加下一次分数,是input变量的multi倍
{addAchievement,input,(multi=1)},增加成就点，同上
{match,input,list,output}input是否在list的列表中
{compare,input1,input2,type,output}比较，支持六种基本关系，输出01
{getHighest,target,output},target取theory/practice/social/achievement等值
()的属性可以不存在，省略
{getLessons,lessontype,symbols,output}获得有指定类型标记的课程列表,两个筛选都可以不选
{selectLessons,input,shownum,selectnum,output}，从input的课程列表中，抽shownum张，选择selectnum（默认1）张
{addToDeck,input}将指定牌加入卡组
{sReward,(shownum=3),(selectnum=1),(lessontype),(symbols)}上述三个的统合版本
{giveProject,name}将指定名称的一张项目牌放到场上，生效
{findProfect,name}引用一张指定名称的项目牌
{giveProgress,num,(type)}获得项目进度，type代表项目类型，匹配的项目进度*2
{giveProgressOn,num,(target)/(name)}为指定卡牌获得项目进度，type代表项目类型，匹配的项目进度*2
{givePermanent,name}将指定名称的一张成果牌放到场上，生效
{drawEvent,num}抽指定数量的事件牌
{addReview,input}增加指定数量的温习值
{deleteLessons,num}删除num张牌库中的课程，


{addsymbol,input,(multi=1)}增加指定的theory/practice/social符号，input*multi个
